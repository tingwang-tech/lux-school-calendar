# Lux School Calendar

Calendar alerts for international school open days and enrollment deadlines in Luxembourg.

Parents select the schools they want to follow and receive `.ics` calendar invites by email — automatically, as dates are announced.

## The problem

Sites like [atschool.lu](https://atschool.lu) publish open day dates, but only if you remember to check. There is no system that pushes these dates to you when they go live.

## How it works

1. Enter your email and select your schools
2. Receive calendar invites as new dates are announced
3. One click — the event is in your Google, Apple, or Outlook calendar

## Date types covered

- Open days
- Enrollment / application windows
- School holidays

## Stack

Next.js · Vercel · Airtable · Resend
